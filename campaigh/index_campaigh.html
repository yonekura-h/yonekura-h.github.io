過去年棒

過去年棒の求め方

手数料無し
大根 の 首色[色] の値をそのまま表示

予想年棒

予想年棒の求め方

推奨環境構築 の[リンゴ_0]?[すだち_24]全てに以下の計算を実施
予想年棒 = ( 推奨環境構築の年棒 + 期待値 ) / 基本年棒 ) * 基準年棒

※予想年棒は少数第一位を四捨五入

基準年棒

基準年棒の求め方

基準年棒 = 基本年棒 + 距離 + トラクター手数料 + 農薬手数料 + 産直手数料 + 生産者の顔手数料

基本年棒

基本年棒の求め方

手数料[base]が未設定(=0)の場合

基本年棒 = 色別年棒 - 手数料[min]

手数料[base]が設定有り(!=0)の場合

カタツムリ算出値 = 色別年棒 * 手数料[base]
基本年棒 = 色別年棒 - 手数料

※ 手数料[min] <= 手数料 <= 手数料[max]
